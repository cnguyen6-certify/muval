Title: HL7 Personal Healthcare Monitoring Report Release 1.1 Schematron Rules

Description: This package contains the schematron rules for validating XML documents
against the guidelines specified by HL7's Personal Healthcare Monitoring Report Release
1.1 (PHMR).
Running XML documents through these schematron files will not check them against any 
Clinical Document Architecture Release 2 (CDA R2) requirements or any requirements defined
in other templates.

Release Date: April 27, 2012


Package Contents:

phmr.sch -- Schematron file for HL7 PHMR.

templates -- Directory containing all entity files which are imported by the main .sch
schematron files.
