The schematron for the BPPC was written according to the constaints defined in the 
IHE IT Infrastructure Technical Framework, Revision 7.0  Vol3 (ITI TF-3): Transactions Section
5.1 Basic Patient Privacy Consent Modules.

A Basic Patient Privacy Consents document is defined to be a CDA and must first validate
to the CDA schema.

The schematron rules are for the following templates:
1.3.6.1.4.1.19376.1.5.3.1.1.1  Medical Document Content Module
1.3.6.1.4.1.19376.1.5.3.1.1.7 - BCCP - With no Scanned Document Part
1.3.6.1.4.1.19376.1.5.3.1.2.6  Consent Services Event
1.3.6.1.4.1.19376.1.5.3.1.1.7.1 BPPC - With Scanned Document
1.3.6.1.4.1.19376.1.5.3.1.2.20 XDS-SD Document
1.3.6.1.4.1.19376.1.5.3.1.2.20.1 XDS-SD Original Author
1.3.6.1.4.1.19376.1.5.3.1.2.20.2 XDS-SD Scanning Device Author
1.3.6.1.4.1.19376.1.5.3.1.2.20.3 XDS-SD Scanner Operator
